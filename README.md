# FitnessTracker


## Installation and Setup

Follow these steps to set up the FitnessTracker project on your local machine.

### Prerequisites

Ensure you have the following installed:
- .NET 6 SDK
- Entity Framework Core
- SQL Server or any other supported database
  
Once installed:
- Check the db connection in `appsettings.json` and change it to your local
- Apply the migrations to set up the database schema
