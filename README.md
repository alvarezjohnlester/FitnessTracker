# FitnessTracker
 This project is built using .NET 6, Entity Framework, and xUnit for testing.

## Installation and Setup

Follow these steps to set up the FitnessTracker project on your local machine.

### Prerequisites

Ensure you have the following installed:
- .NET 6 SDK
- Entity Framework Core
- SQL Server or any other supported database
  
Once installed:
- Check the db connection in `appsettings.json` and change it to your local
- Apply the migrations to set up the database schema


### Technologies Used

- **.NET 6**: The main framework used for building the application.
- **Entity Framework Core**: ORM for database operations.
- **xUnit**: Testing framework for unit tests.
- **Moq**: Library for creating mock objects in unit tests.
- **AutoMapper**: Object-object mapper to map DTOs to entities and vice versa.
- **SQL Server**: Database for storing application data.
- **ASP.NET Core**: Framework for building web APIs.
- **Microsoft.Extensions.Logging**: Logging framework for logging application activities.
